# LearnLitecoin redesign starter

This package is a static multi-page starter for a stronger LearnLitecoin website focused on two outcomes:

1. teaching people how Litecoin works
2. helping merchants accept Litecoin through a BitPay-first rollout

## Files included

- `index.html` — homepage
- `start-here.html` — beginner onboarding page
- `wallets.html` — wallet guide page
- `accept-litecoin.html` — merchant / BitPay page
- `mweb.html` — privacy page
- `contact.html` — intake / CTA page
- `styles.css` — shared site styling
- `script.js` — simple reveal animation

## Before you publish

Replace these demo items first:

- `hello@learnlitecoin.com`
- merchant package wording if needed
- any wallet recommendations you want changed
- forms or buttons that should point to your real contact form or scheduler

## How to upload to GitHub in the web interface

1. Sign in to GitHub.
2. Create a new repository named something like `learnlitecoin-site`.
3. Open the new repository.
4. Click **Add file** then **Upload files**.
5. Drag all files from this folder into the upload area.
6. Commit the upload to the `main` branch.

## Suggested structure if you use Git locally

```bash
git init
git add .
git commit -m "Initial LearnLitecoin redesign"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/learnlitecoin-site.git
git push -u origin main
```

## Important Squarespace note

If your current LearnLitecoin site is on Squarespace and you plan to use the Developer Platform, make a backup or duplicate first. Switching to developer mode replaces the current template with an editable developer template and stops normal template updates from Squarespace.
